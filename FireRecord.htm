FireRecord

FireRecord is a lightweight firefox extension to record and playback user driven web events, to simulate web browsing for automation or testing purposes
Some features include:

  • Record
    Append the active script with user event
  • New Script
    You can create a new script, and play it back.
  • Play
    Execute the script to simulate the user events

    • FireRecord has only been tested using Firefox 20.0.1
